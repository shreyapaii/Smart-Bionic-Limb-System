#Smart Bionic Limb System
AI-powered EMG gesture recognition system for low-cost prosthetic limb research.

Built using Machine Learning, EMG signal processing, and Streamlit to predict hand gestures in real time and monitor rehabilitation progress through an interactive healthcare dashboard.

Features:
Real-time EMG gesture prediction
Automatic preprocessing pipeline
Multi-model ML training (XGBoost, Random Forest, LightGBM)
Prosthetic hand visualization
Patient rehabilitation dashboard
Session tracking and analytics
Interactive Streamlit UI

Tech Stack:
Python
Streamlit
Scikit-learn
XGBoost
LightGBM
Pandas
Plotly
